# Custom JS Widget for Elementor

A premium WordPress plugin that adds a Custom JavaScript Widget to Elementor, allowing for raw JS injection with placement control (Inline, Header, Footer).

## Features
- **CodeMirror Editor**: Syntax-highlighted JS input.
- **Multiple Placements**: Choose between Inline, Head, and Footer.
- **Dependency Management**: Toggle jQuery requirement.
- **Auto-Updates**: Integrated GitHub version checker.
- **Automatic Releases**: CI/CD pipeline for ZIP packaging.

## Requirements
- WordPress 6.0+
- PHP 8.2+
- Elementor
